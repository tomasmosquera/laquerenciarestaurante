Conexión con GitHub
Los cambios que guardes se escribirán directamente en tu repositorio de GitHub. Vercel detectará el commit y redesplegará el sitio en ~1–2 minutos.
Cómo obtener el token:
1. Ve a github.com → Settings → Developer Settings → Fine-grained tokens
2. Crea un token con acceso a este repositorio y permiso Contents → Read and write
3. Copia el token (empieza por github_pat_…) y pégalo abajo.
Se guarda solo en tu navegador (localStorage), nunca se envía a ningún servidor propio.
